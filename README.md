## ğŸ“Œ Overview

The **SVSU Dudhola Campus Navigation System** is a **front-end web-based navigation solution** that helps students, faculty, and visitors easily locate campus buildings and navigate between them using an **interactive map interface**.

The system visually represents the **SVSU campus layout** and calculates optimal paths between locations using **graph-based shortest path algorithms**, similar to real-world navigation systems like **Google Maps**.
